## Gen5 설계 규칙

### 규칙 1: cyano+F 조합 유지 (검증된 hERG 해결사)
- F: LogP 조절 + 대사 안정성
- CN: 극성↑ → hERG↓, 도킹 친화도↑
- → 이 조합을 핵심으로 유지하되 위치/수량 변형

### 규칙 2: MD 안정성 올리는 구조 전략
- 고리 수 증가 → 강직성↑ → RMSD↓
- 링커에 메틸 추가 → 유연성↓
- 스피로 구조 탐색 → 아다만탄 특성 강화

### 규칙 3: ether_direct 링커 유지 + 변형
- hERG에 가장 유리한 링커 (검증됨)
- 다만 너무 짧아 MD 불안정 → 링커에 메틸 추가 버전 시도

### 규칙 4: sulfonyl 계열도 유망
- hERG 0.304 (최저) → DILI만 더 낮추면 됨
- sulfonyl + cyano 이중 치환 탐색 가치 있음

## Gen5 설계 지시

### 전략: "cyano+F 유지 + 강직성 강화로 MD 안정성 확보"

**방향 1: 고리 강직성 강화 (핵심)**
- cyano_fluorophenyl → 나프탈렌 계열로 확장 (고리 2개)
- cyano_fluorophenyl + 메틸 추가 (입체장애로 유연성↓)
- indanyl 계열 (5+6 융합 고리 → 강직성↑)

**방향 2: cyano+F 위치 최적화**
- 2F-4CN (오르토 F, 파라 CN)
- 3F-5CN (메타 위치 조합)
- 4F-2CN (현재와 반대)

**방향 3: 링커 미세 조정**
- ether_methyl (O-CH2- 추가) → 약간의 유연성 완충
- ether_direct + gem-dimethyl (링커에 메틸 두 개)

### 목표 수치 (Gen5)
- hERG < 0.4 ✅ (Gen4에서 달성, 유지)
- DILI < 0.4 ✅ (유지)
- MD RMSD < 2.5Å ← 핵심 목표 (Gen4 2.66Å에서 개선)
- 도킹 ΔG < -4.5 ✅ (유지)
- SA Score < 5.5 ✅ (유지)
