# Epsilon-Gate 설계 철학 및 비전
## Philosophy & Vision Document

> 이 문서는 Gen0 수작업부터 Gen12까지의 과정에서 형성된  
> 설계 철학과 미래 방향성을 기록한 것입니다.

---

## 핵심 철학: 약 = 독

> *"약은 독이다. 적소에 적당량을 쓰는 것이 최선이다."*

이 전제가 Gen0부터 Gen12까지 모든 설계 결정에 녹아있습니다.

대부분의 신약 개발이 **효능 극대화**를 목표로 하는 반면,  
이 프로젝트는 처음부터 **독성 최소화** 위에서 효능을 찾았습니다.

```
일반적 접근:  효능 최대화 → 독성 나중에 확인
이 프로젝트:  독성 최소화 → 그 안에서 효능 찾기
```

Gen11 최종 후보가 hERG=0.14, DILI=0.025를 달성한 것은  
우연이 아니라 **철학이 데이터로 나타난 결과**입니다.

## 치료 전략: 3단계 순서

### 1단계 — 브릿지 치료 (증상 완화)
- 부작용 최소 + BBB 우수 + 천연물 시너지 가능
- 뇌 환경을 안정화하는 것이 목적
- Gen11 direct_O_pyrrolidine이 이 포지션

### 2단계 — 신약물질 (증상 개선)
- 레카네맙 같은 타겟 치료
- **1단계로 뇌 염증이 줄어든 상태**에서 투여
- 부작용 감소 + 효과 증가 시너지

### 3단계 — RNA 치료 (근본 해결)
- 유전자 수준에서 원인 차단
- 1, 2단계가 만든 안정된 환경에서 효과 극대화

**순서가 중요합니다.**  
레카네맙이 부작용이 큰 이유 중 하나는 뇌가 이미 염증 상태이기 때문입니다.  
불난 집에 소방차가 들어가면 2차 피해가 납니다.

## 천연물 시너지

오메가3, 커큐민, 블루베리 등이 알츠하이머에 효과적인 이유:

```
Gen11 (브릿지 치료제)
  → BBB 0.90으로 뇌에 도달
  → AChE 억제로 신경전달 안정화
  → 뇌 환경 안정화
      ↓
커큐민 / 오메가3 / 블루베리
  → 항염증
  → 아밀로이드 억제
  → 신경 재생 (BDNF)
      ↓
더 나은 치료 효과
```

Gen11이 뇌 환경을 정비하면 천연 성분들이 그 안에서 더 잘 작동합니다.

---

## 치료 지수 (Therapeutic Index)

도네페질 IC₅₀ = 11nM — 강력하지만 부작용도 강합니다.

Gen11이 IC₅₀ = 50~200nM 범위가 나온다면:
- 도네페질보다 약하게 보일 수 있음
- 그러나 BBB 0.90으로 뇌 도달량 자체가 많아짐
- 더 낮은 농도로 같은 효과 가능
- **치료 지수(효과 농도 vs 독성 농도 간격)가 넓어짐**

효능 숫자가 아니라 **치료 창의 넓이**가 좋은 약의 기준입니다.

## 방법론: 병목 해결 엔진

파레토 탐색보다 효율적인 접근:

```
병목 감지
  → 병목 이전 단계 최선값 유지
  → 병목 단계만 집중 최적화
  → 통과하면 다음 단계 진행
```

Gen10에서 MD 안정성이 병목임을 감지하고,  
독성 최선값(Gen8)을 유지하면서 링커만 최단화(Gen11)해  
처음으로 4지표를 동시에 달성했습니다.

## AI 활용 방법론

### 엔진 vs 파이프라인

**파이프라인:** 설계된 경로를 흘러가는 것  
**엔진:** 실패 데이터에서 패턴을 뽑아 다음 설계를 바꾸는 것

```
실패 데이터 → 패턴 추출 → 설계 수정 → 재실행
      ↑__________________________________|
```

### 방향 에이전트로서의 인간

```
인간 (방향 에이전트):
  목적 설정
  병목 감지
  철학적 판단
  "무엇이 좋은 약인가"

AI (실행 엔진):
  코드 실행
  데이터 분석
  패턴 추출
  결과 계산
```

AI가 잘 아는 영역에서는 AI가 이끌고,  
AI가 모르는 영역에서는 사람이 이끕니다.  
그 경계를 아는 것이 AI를 제대로 쓰는 것입니다.

### Claude 내부 벡터 활용

논문이 "방향"을 제공하고,  
Claude 내부 벡터(수십 년 의약화학 지식)가 "구현"을 제공합니다.

```
논문 → 방향 확인 (최소 읽기)
Claude 벡터 → 구현 + 패턴 추출
실패 데이터 → 다음 설계 반영
```

## 미래 비전

### 신약 개발 압축

```
현재:  신약 개발 10년 + 1조원
예정:  Mythos5 + 디지털 트윈 → 2~3년 + 수백억
```

### 사전대응 의학 시대

```
현재 패러다임:  증상 발현 → 진단 → 치료 (사후대응)
미래 패러다임:  유전자 스크리닝 → 위험군 발견 → 예방 투여 (사전대응)
```

Gen11의 특성(BBB 우수 + 독성 최소)은  
**장기 예방 투여**에 맞는 구조입니다.

### 희귀질환 커버

```
현재:  7000개 희귀질환 중 대부분 치료제 없음
       (임상 비용 1조원 → 수익 안 됨)

압축 후: 수백억으로 임상 가능
       → 희귀질환 전부 커버 가능
```

## Gen0 → Gen12 핵심 교훈

| 세대 | 교훈 |
|------|------|
| Gen0 | 독성을 먼저 보는 것이 맞다 |
| Gen3 | 도킹 점수 ≠ MD 안정성 |
| Gen4 | cyano+F = hERG 해결사 |
| Gen5 | F 두 개 = 도킹 희생 (트레이드오프) |
| Gen6 | F 위치 하나가 activity cliff 만든다 |
| Gen7 | 승인약 구조가 최고의 시드 |
| Gen8 | 하이브리드 = 균형점 |
| Gen10 | 병목을 찾아라, 이전 최선값을 지켜라 |
| Gen11 | 링커 하나 줄이면 세계가 달라진다 |
| Gen12 | N-방향족 = hERG 상승, 탄소-CN = hERG 유지 |
